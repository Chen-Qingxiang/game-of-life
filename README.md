# Conway's Game of Life

A lightweight interactive GitHub Pages project for Conway's Game of Life, built with plain HTML, CSS, and JavaScript.

The visualizer uses a canvas grid, standard B3/S23 rules, editable cells, pattern presets, speed control, grid sizing, and random density control.

## Preview locally

Open `index.html` directly in a browser, or serve the folder with a small local static server:

```bash
python3 -m http.server 8081
```

Then visit:

```text
http://localhost:8081/
```
